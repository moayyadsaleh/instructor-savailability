Instructor Teaching Schedule Manager

How to Use

1. Add a Class: Enter the instructor's name, the class name, and select a start and end date/time. Click "Add Class". The class schedule will appear on the calendar immediately.

2. View Details: Hover over a class slot to see more details in a pop-up.

3. Delete a Class: Click on a class slot and confirm the deletion to remove it from the calendar.
